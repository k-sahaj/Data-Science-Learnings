# Mastering SQL for Data Science

A comprehensive roadmap and resource collection for learning **SQL from scratch to an advanced level**, with a strong focus on **Data Science, Analytics, and Interview Preparation**.

---

## Why Learn SQL?

### For Data Science

* the not so evolved Data ecosystem (you will have to extract data yourself. the data source is database itself.)
* Performing EDA, cleaning and transforming data
* Building datasets for ML modeling
* Understanding business data structures

### Industry Demand

SQL consistently ranks among the most sought-after technical skills across data-related roles.
Almost 80% companies use SQL based Database.
SQL is the most required skill from a job candidate right now.

## Recommended Database

### Start With: MySQL

MySQL is beginner-friendly and widely used. And also FREE.

Once you understand SQL fundamentals, transitioning to other databases such as PostgreSQL, SQL Server, or Oracle becomes much easier since most SQL concepts remain similar. One can master them when required.

# SQL Fundamentals

## SQL Categories

### DDL (Data Definition Language)

Used to define and modify database structures.

```sql
CREATE
ALTER
DROP
TRUNCATE
```

### DML (Data Manipulation Language)

Used to work with data stored inside tables.

```sql
SELECT
INSERT
UPDATE
DELETE
```

### DCL (Data Control Language)

Used to manage permissions.

```sql
GRANT
REVOKE
```

### TCL (Transaction Control Language)

Used to manage transactions.

```sql
COMMIT
ROLLBACK
```
As we are DS oriented, we'll focus more on DML part coz that will be the most we will be using in practice.

* GROUP BY
* HAVING Clause

* Joins

  * FULL JOIN
* Subqueries
* Temporary Tables
* CASE Statements
* Views

## Advanced Level

* Stored Procedures
* User Defined Functions (UDFs)
* Common Table Expressions (CTEs)
* Window Functions
* Query Optimization
* Performance Tuning
